# Gamified Todolist

A simple, retro app that gamifies your to-do list, inspired by Duolingo

(Work in progress)

### PERHATIAN ⚠️

Project ini dibuat untuk Lomba FESTIKA 2025 atas nama tim Zona Inovasi dan hanya bersifat sebagai prototype.

Basis kode ini dapat berisi kode, komentar, atau dokumentasi yang dihasilkan oleh AI. Meskipun kami berupaya menjaga keakuratan, kualitas, dan keamanan, seluruh materi disediakan "sebagaimana adanya" dan mungkin mengandung kesalahan atau kekurangan. Harap meninjau, menguji, dan memvalidasi seluruh kode sebelum digunakan dalam lingkungan produksi. Konten yang dihasilkan AI mungkin tidak mencerminkan peninjauan pakar manusia, praktik terbaik, atau opini pribadi para pengelola repositori ini.
